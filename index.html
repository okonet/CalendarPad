CalendarPad by okonet

A Mootools based class for a 3 months date selector.

CalendarPad allows you to pick a date from a 3-months calendar.

Demo

Dependencies

  • Mootools Core 1.2.4
  • Mootools More 1.2.4
    • Date
    • Hash.Extras
    • String.Extras
    • Element.Delegation

Install

Download, include mootools.js and calendarpad.js & *.css files. See example.html for more info.
